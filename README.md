# ToDo App

Bu projede, Core Data kullanarak yapılacaklar listesi uygulaması geliştirildi. Uygulama, kullanıcıların görevlerini takip etmelerine, tamamlanan görevleri işaretlemelerine ve yeni görevler eklemelerine olanak tanır.
